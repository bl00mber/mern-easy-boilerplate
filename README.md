# MERN-easy-boilerplate

* Mongoose
* Express.js
* Redux
* Node.js
* Heroku
* Stylus
* Material-UI

## Commands

Localhost: [http://localhost:3000](http://localhost:3000)

``` bash
# install dependencies
$ npm i

# run with client-side hot reload
$ npm start

# build for production with minification
$ npm run build

# production run
$ npm run start:prod
```

``` bash
# run with server-side hot reload
$ npm start:demon
```
